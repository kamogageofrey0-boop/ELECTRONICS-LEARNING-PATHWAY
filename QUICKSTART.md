# 🚀 Quick Start Guide

## Get Started in 3 Simple Steps

### 1️⃣ Navigate to App Directory
```bash
cd electronics_app
```

### 2️⃣ Install Flask (if needed)
```bash
pip install flask
```

### 3️⃣ Run the Application
```bash
python app.py
```

### 4️⃣ Open Your Browser
Navigate to: **http://127.0.0.1:5000**

---

## 📖 First Time User Guide

### What You'll See

**Dashboard:**
- Progress overview (0% initially)
- 5 learning modules
- Module 1 marked as "Current Module"
- Other modules locked 🔒

### How to Learn

1. **Click "Continue Learning"** on Module 1
2. **Read** the key concepts and notes
3. **View** diagrams and images (click to enlarge)
4. **Watch** video tutorials (click thumbnail to play)
5. **Answer** the questions below the content
6. **Submit** your answers
7. **Get** immediate feedback
8. **Click** "Continue to Next Module" when all correct
9. **Repeat** for all 5 modules

> - A) Volts
> - B) Amperes
> - **C) Watts** ✓
> - D) Ohms

**Numerical Input:**
> "If a device uses 100J of energy in 5 seconds, what is its power consumption in Watts?"
> Answer: **20** (Power = 100J / 5s = 20W)

## ❓ Troubleshooting

### Videos Not Loading
- Check internet connection
- Allow YouTube embeds in browser
- Try different browser
- Clear cache

### Can't Submit Answers
- Make sure all questions answered
- Check number format (no text in numeric fields)
- Refresh page if stuck
- Try different browser

### Progress Lost
- Progress stored in session (temporary)
- Clears when server restarts
- Normal behavior for demo
- Use database for production
